Key Features of a Bullet Journal


Content, Layout and Format

Bullet Journals are a tool for helping you stay organised and keep your life on track, so it only makes sense that there is an order to the Bullet Journal!

Contents

All Bullet Journals contain the same basic pages

  • An index page
  • Log pages
  • A migration page
  • An overview page
  • Tracker pages
  • Collection pages

Layout

The index page should be at the very start of your Bullet Journal. The purpose of the index page is to act like a contents page, detailing where to find everything in your journal so when you want to add something to a list or fill in you're weeks page you know where to look!

Your index page can also include a key, if you decide to use one, for your trackers or lists to symbolise what the tasks mean i.e. important, complete, etc...
